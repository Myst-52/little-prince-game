# 小王子 · 星际漫游

> "所有的大人都曾经是小孩，虽然，只有少数的人记得。" —— 安托万·德·圣-埃克苏佩里

## 项目简介

这是一个基于经典文学作品《小王子》的网页文字冒险游戏。玩家将跟随小王子的视角，体验一段关于爱、友谊与成长的星际旅程。通过做出不同的选择，玩家可以探索不同的剧情分支，最终获得独特的结局体验。

## 创意构思

### 设计理念

1. **诗意与交互的结合**：将《小王子》原著中的诗意语言与现代网页交互技术相结合，创造沉浸式的阅读体验。

2. **选择的重量**：每个选择都会影响故事走向和最终结局，让玩家体会到"选择"的意义。

3. **视觉叙事**：通过星空背景、流动的动画和温暖的配色，营造梦幻般的童话氛围。

### 故事结构

游戏采用**分支叙事**结构，包含：
- 1个序章 + 多个章节
- 15+ 个故事场景
- 4 个独特结局
- 多条探索路径

### 核心主题

- **爱与责任**：通过玫瑰的故事探讨爱的真谛
- **友谊与羁绊**：玩家与小王子之间的情感联结
- **成长与回归**：从童真到成熟再到回归本心

## 技术方案

### 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构与语义化标签 |
| CSS3 | 视觉设计、动画效果、响应式布局 |
| JavaScript (ES6+) | 游戏逻辑、状态管理、DOM操作 |

### 核心特性

#### 1. 星空背景系统
- 使用 JavaScript 动态生成 150+ 颗星星
- CSS 动画实现星星闪烁效果
- 流星动画增加视觉层次

#### 2. 故事引擎
- 基于 JSON 的场景数据结构
- 支持分支选择和状态追踪
- 自动打字机效果的文本展示

#### 3. 状态管理系统
- 羁绊值（Love）：记录与小王子的情感深度
- 星星（Stars）：探索奖励
- 物品（Items）：收集要素

#### 4. 响应式设计
- Mobile-first 设计理念
- 支持桌面端和移动端
- 适配不同屏幕尺寸

## 使用方式

### 本地运行

1. 解压压缩包
2. 用浏览器打开 `index.html` 文件
3. 点击"开始旅程"按钮开始游戏

### 游戏操作

- **点击按钮**：做出选择，推动故事发展
- **观察数值**：注意羁绊值和星星的变化
- **探索路径**：尝试不同的选择，发现不同的结局

### 游戏结局

1. **回家结局**：小王子与玫瑰重逢
2. **友谊结局**：与飞行员成为永恒的朋友
3. **珍藏结局**：珍藏与小王子共度的时光
4. **真理结局**：领悟爱与自由的真谛
